This theme is a sleek and modern implementation of Ubuntu SO design.

## Components:
- Banner with title
- "About Me" section
- "Skills" section
- "Projects" section
- Contact form (Programming logic for sending the form is not applied).
- Blog section.
- "About me" section more extended.
